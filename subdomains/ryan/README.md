# ryan.war.re — resume

Next.js app served at [ryan.war.re](https://ryan.war.re).

## Dev

```bash
yarn install
yarn dev      # localhost:3000
yarn lint
yarn test
yarn build
```

## Notes

- Pages router (`pages/`)
- `/` is rewritten to `/n` via `next.config.js`
- Path alias `@/*` → project root
- Jest + Testing Library for component tests
